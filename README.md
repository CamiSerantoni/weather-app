# 🌦️ Weather Finder

![React](https://img.shields.io/badge/React-18-blue)
![API](https://img.shields.io/badge/API-OpenWeather-orange)
![License](https://img.shields.io/badge/license-MIT-green)

Aplicación web desarrollada con **React** que permite consultar el estado del tiempo en cualquier ciudad utilizando la API de **OpenWeatherMap**.

La aplicación muestra información meteorológica en tiempo real para ayudar a los usuarios a **planificar sus actividades diarias según las condiciones climáticas**.

## ✨ Features

- 🔍 Consulta del clima por **ciudad y país**
- 🌡️ Visualización de **temperatura actual**
- ☁️ Información meteorológica en tiempo real
- ⚡ Consumo de **API REST OpenWeatherMap**
- ⚛️ Manejo de estado con **React Hooks**
- 📱 Interfaz **responsive**
- ❗ Validación de datos ingresados por el usuario
- 🚨 Manejo de errores cuando la ciudad no existe


---

## 🛠️ Tecnologías utilizadas

- **React**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **Materialize CSS**
- **OpenWeatherMap API**

Estas tecnologías permiten construir una aplicación web interactiva que consume datos meteorológicos en tiempo real.
